# ProtonOS
ProtonOS to system operacyjny oraz botloader opaty na asembler.

## Zastosowanie ProtonOS
Cześć jestem @bugs_game pasjonóje się tworzeniem aplikacji webowych chciał bym z wami pozielic się moją przygodą z innym językiem programowania niż html, css czy javascript i tak dalej.
Czemu możesz chcieć zobaczyć ProtonOS jest to bardzo lekki system który może służyć do serwerów np. do hostowania stron cz nawet botów python.

## Zamknięte beta-test
Beta-testy na razię są zamknięte możesz się zgłosić do @BugsGame aby dołączyć.

## SDK dla developerów
Jeśli chcesz nas wspierać programówjąc razem z nami możesz skorzystać z repo na moim profilu które tłumaczy wymagania itp.

## Uwaga!
ProtonOS jest niestabilny oraz może zniszczyć sprzęt niezaleca się insalowania na prawdziwy komputer.
